# untangle

Tensor decoupling in Jax. Collection of algorithms for decoupling multivariate functions using tensor decompositions.

This project was built using `uv` (https://docs.astral.sh/uv/). 

To run the tests: `uv run -m unittest -v`.

### Algorithms

- "Basic" Tensor Decoupling Algorithm | [link](https://arxiv.org/pdf/1410.4060)
- Tensor Decoupling with Polynomial Constraints | [link](https://researchportal.vub.be/en/publications/multivariate-polynomial-decoupling-in-nonlinear-system-identifica)
- CMTF with B-Splines Projection | [link](https://arxiv.org/pdf/2504.03263)
- CMTF with Smoothing Splines Projection
